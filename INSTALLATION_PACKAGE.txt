================================================================================
                           QUICK START (LOCAL)
================================================================================

STEP 1: Clone the Repository
   git clone https://github.com/FreecoDAO/OpenManus.git
   cd OpenManus

STEP 2: Install Dependencies
   pip3 install fastapi uvicorn pydantic aiohttp

STEP 3: Configure Minimax API Key
   - Edit .freeco/settings.json
   - Add your Minimax M2 Pro API key

STEP 4: Start the Server
   ./run_gui.sh
   OR
   python3 -m uvicorn app.gui_server:app --host 0.0.0.0 --port 8000 --reload

STEP 5: Access the GUI
   Open browser: http://localhost:8000

API DOCUMENTATION

================================================================================
                          TROUBLESHOOTING
================================================================================

Port 8000 already in use?
  python3 -m uvicorn app.gui_server:app --port 8001

Minimax connection fails?
  1. Verify API key in Settings tab
  2. Check internet connection
  3. Use Test Connection button

Tasks not saving?
  1. Check .freeco/ directory exists
  2. Verify write permissions
  3. Check disk space

GUI not loading?
  1. Check server is running
  2. Try http://localhost:8000/docs
  3. Check browser console for errors
